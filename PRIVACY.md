# Privacy Policy

**Last Updated: February 26, 2026**

This project is built with a **Privacy-First** and **Offline-First** philosophy. Because this is an open-source desktop application, we believe you should have total control over your business data.

### 1. Data Storage
All data managed by this ERP is stored locally on your device in a **SQLite database**. 
* We do not own, manage, or have access to your database file.
* Your data never leaves your machine unless you manually export it or move the database file yourself.

### 2. No Telemetry or Tracking
* **No Analytics:** We do not track how you use the app.
* **No "Phone Home":** The application does not connect to any external servers to report usage, crashes, or user behavior.
* **No Accounts:** You do not need to "Log In" via a cloud service to use the software.

### 3. Internet Connectivity
The application is designed to function entirely **offline**. Internet access is only required if:
* You are downloading updates via GitHub.
* You choose to integrate third-party web services (which will be clearly labeled).

### 4. Transparency
As an **Open Source** project under the **GPLv3 License**, the source code is available for public audit. Anyone can inspect the code to verify that no data-collection mechanisms exist.

**Your business is your business.**
